About

Project for showing of different features of PWA and its technologies. This site uses all sorts of methods and technologies of progressive APPs to show tools that developers of sites have in their hands.

Manifest

Web app manifest is a W3C specification defining a JSON-based manifest.

WebAssembly

WebAssembly allows precompiled code to run in a web browser, at near-native speed. In this site I used Python integration based on Pyodide and imported such libraries as: Plotly, Pandas, python-osc, SocketIO and so on.

Service worker

A service worker is a web worker that implements a programmable network proxy that can respond to web/HTTP requests from the main document. It is able to check the availability of a remote server, cache content when that server is available, and serve that content later to the document. Service workers, like any other web workers, work separately from the main document context.
